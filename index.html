html 复习

盲点复习

  • <strong>和<b>标签都可以使文本以粗体展示标签中的文本，但是<strong>标签的语义是标签中的内容具有很高的重要性，而<b>标签的语义仅仅是加粗文本以引起读者的注意，并没有特殊的意思。
  • <em>和<i>标签默认情况下均以斜体显示标签中的文本，但是<em>标签具有强调的语义，用来表示标签中的内容很重要，而<i>标签仅仅用于定义斜体文本。
  • 浏览器根据历史记录来判断超链接是否被点击过，如果 href 属性和历史记录中的某条 URL 重合，那么说明该链接被点击了。清空浏览器的历史记录会让超链接的颜色再次变回蓝色。
  • HTML5 中增加了<picture>标签
  • 1*.在 <picture> 标签中包含零个或多个 <source> 标签，通过 <source> 标签中的 media 属性可以设定匹配条件（允许哪个版本的图片显示），通过 srcset 属性可以定义图片的路径。另外，在 <picture> 标签的最后还需要定义一个 <img> 标签
  • 2*.浏览器将评估每个 <source> 标签，并在其中选择最合适的 <source> 标签，如果未找到匹配项，则使用<img> 标签所定义的图片。另外，<img> 必须是\ 标签的最后一个子元素。
  • 1.<caption> <table> <tr> <th> <td>
    2.属性：rowspan 属性表示向下合并单元格，colspan 属性表示向右合并单元格。
    3.每次合并 n 个单元格都要少写 n-1 个<td>标签。
    4.即使一个单元格中没有任何内容，我们仍需使用 或 元素来表示一个空单元格的存在，建议在 或 中加入  （空格），否则低版本的 IE 可能无法显示出这个单元格的边框。
  • <dl> <dt> <dd>
  • <form> action 属性用来指明将表单提交到哪个页面；
    method 属性表示使用哪个方式提交数据，包括 GET 和 POST 两种方式，它们两者的区别如下：
  • 1*.GET：用户点击提交按钮后，提交的信息会被显示在页面的地址栏中。一般情况下，GET 提交方式中不建议包含密码，因为密码被提交到地址栏，不安全。
  • 2*.POST：如果表单包含密码这种敏感信息，建议使用 POST 方式进行提交，这样数据会传送到后台，不显示在地址栏中，相对安全。
  • *3.**<label> 为表单中的各个控件定义标题
  • *4.**<input> 定义输入框
  • *5.**<textarea> 定义文本域（一个可以输入多行文本的控件）
  • *6.**<select> 定义下拉列表
  • *7.**<option> 定义下拉列表中的选项
  • *8.**<button> 定义一个可以点击的按钮
  • 嵌入css：链接式<**link** href=”名称” type=”text/css” rel=”stylesheet” />
    导入样式使用 @import 命令导入外部样式表。
    <**style** type=”text/css”> @import url(daoru.css);</**style**>
  • html布局：
  • *1.**<nav> 用于定义网页中的导航栏。
  • *2.**<section>代表 HTML 文档中的“节”或“段”，“段”可以理解为一篇文章里按照主题的分段，“节”则可以理解为一个页面里的分组。其主要作用就是对页面的内容进行分块或者对文章的内容进行分段。
  • *3.**<article> 用于定义文章或者其它独立的信息，代表一个页面中自成一体的内容，例如论坛的帖子、博客上的文章、一篇用户的评论等。
  • *4.**<aside> 用于定义网页内容以外的部分，例如网页的侧边栏。
  • *5.**<footer> 用于定义网页的底部，例如作者、版权等信息。
  • *6.**<details> 用于定义一些详细信息，并且可以根据需要隐藏或显示这些详细信息。
  • *7.**<summary> 用于为<details>标签定义标题。
  • *8.**<header>用于定义网页的头部，头部中一般包含一些介绍性的内容，例如网站名称、logo 或者作者的信息。
  • 头部<head>标签
  • 1.<title>* 标签用来定义 HTML 文档的标题，只有包含 <title> 标签的文档才算是一个有效的 HTML 文档。另外，一个 HTML 文档中仅允许存在一个 <title> 标签，并且 <title> 标签必须放置在 <head> 标签中。
  • 2.<base>* 标签用于为页面中所有相对链接指定一个基本链接，当您设置了基本链接后，当前页面中的所有相对链接都会使用这个基本链接作为前缀
  • 3.<link>* 标签经常用于引用外部 CSS 样式表，<link> 标签中包含两个主要的属性，分别是 rel 和 href。rel 属性用来指示引用文件的类型，href 属性用来设置外部文件的路径。
  • *4.**使用 <style> 标签可以在 HTML 文档中嵌入 CSS 样式
