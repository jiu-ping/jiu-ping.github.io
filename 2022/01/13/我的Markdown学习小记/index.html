Markdown 学习小记

Markdown标题

用#递增，至多六个，分为一到六级标题

  • 分割线：—
    （ 一级标题自带分割线）

Markdown 段落格式

  • 换行：末位添加两个空格后回车 

    （也可以直接使用空行重起段落）

  • 字体： 斜体文字 * 或者 _ 

    **粗体文本**   **或者 __  
***粗斜体文本***   ***或者___

  • 删除线：我被删除了 ~~

  • 下划线：我是下划线 使用标签

  • 创建脚注： （暂不成功）

    创建脚注格式类似这样 [^RUNOOB]。

Markdown 列表

  • 有序列表 *

  • 列表嵌套 在子列表中的选项前添加四个空格
